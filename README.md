## 프로젝트 소개
**왜 Yemon을 만들었나요?**
<br/>지인과 함께한 식사 자리나 단체 여행 후에 정산을 편리하게 할 수 있는 서비스의 필요성을 느껴 기획한 정산 앱 개발 프로젝트입니다.

**Yemon은 어떻게 이용하면 되나요?**
<br/>단지 영수증 정보를 입력하고 사람들을 매칭해주면 모든 사람들에 대한 정산 정보가 생성됩니다.
<br/>정산된 결과 이미지를 저장하거나 카카오톡으로 공유해서 정산 결과를 알리세요.

**Yemon을 이용해서 간편하게 정산해보세요.**
